# 📈 Data Analysis & Visualization Tool

## 🚀 About the Project

This project is a command-line based data analysis and visualization tool built using Python.

It allows users to load datasets and perform complete data analysis, including exploration, cleaning, statistics, and visualization.

---

## 🧰 What You Can Do

📂 **Load Dataset**

* Import CSV files dynamically

🔍 **Explore Data**

* View head, tail
* Check column names and data types
* Get dataset info

🧹 **Data Cleaning**

* Identify missing values
* Fill missing values (mean/custom)
* Drop missing data

📊 **Statistical Analysis**

* Generate descriptive statistics
* Analyze dataset structure

📉 **Visualization Options**

* Bar Chart
* Line Plot
* Scatter Plot
* Pie Chart
* Histogram
* Stack Plot

💾 **Save Output**

* Save generated plots as image files

---

## ⚡ How the Program Works

1. Load your dataset (CSV file)
2. Choose operation from menu
3. Perform analysis or visualization
4. View or save results

---

## 📚 Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## ▶️ Run the Program

python "Visualizer.py"
